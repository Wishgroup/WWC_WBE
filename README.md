# Wish Waves Club Website

A modern React website for Wish Waves Club, an exclusive membership platform offering events, discounts, and premium experiences.

## Features

- Responsive design matching the WHOOP website structure
- Custom color palette (teal to gray gradient)
- Membership tiers (Essential, Premium, Elite)
- Event and benefits showcase
- Member testimonials
- Modern React with Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The website will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
  components/
    Header.jsx       - Navigation header
    Hero.jsx         - Hero section with background image
    Memberships.jsx  - Membership tiers
    Features.jsx     - Benefits and features
    Testimonials.jsx - Member testimonials
    Footer.jsx       - Footer with links
  App.jsx            - Main app component
  main.jsx           - Entry point
  index.css          - Global styles
```
